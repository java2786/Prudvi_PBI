# Create venv
```bash
python3 -m venv my_env
```

# Activate environment

### Linux or mac
```bash
source my_env/bin/activate
```

### Windows
```bash
my_env/Scripts/activate
```


# Install NumPy
```bash
pip install numpy
pip install numpy==1.21.0
```
# Install pandas
```bash
pip install pandas
```
