## Answer Keys and Solutions  
  
### Note for Students:  
Try to solve these problems on your own first. The learning happens when you struggle with the problem and find solutions. Use the following approach:  
  
1. **Read the problem carefully**  
2. **Understand the input and expected output**  
3. **Break down the problem into smaller steps**  
4. **Write pseudocode first**  
5. **Then write actual Python code**  
6. **Test with different inputs**  
7. **Debug if errors occur**  
  
### How to Practice:  
- Start with Topic 1 and complete all problems  
- Don't skip to next topic until current is clear  
- Try solving without looking at hints  
- Run your code and test with different inputs  
- If stuck for more than 15 minutes, discuss with classmates or instructor  
- Revisit difficult problems after a day or two
